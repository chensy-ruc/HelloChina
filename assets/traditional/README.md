# Traditional Visual References

本目录存放曾用于 HelloChina 首页中国风装饰的公开 SVG 参考素材，均已本地化，避免运行时依赖外部热链。

当前页面的正式装饰已经改为 CSS 绘制的折扇、如意云、中国结、玉璧与青花水波元素，并以低层级背景纹理的方式嵌入页面，不再作为独立图标堆叠在内容板块里。下列 SVG 保留为公开素材来源记录和后续设计参考。
