# Traditional Visual Assets

本目录存放 HelloChina 首页中国风装饰元素，均为本地化 SVG 文件，避免运行时依赖外部热链。

## 素材来源

- `plum-blossom.svg`：OpenClipart，public domain，源文件下载地址：`https://openclipart.org/download/185023/plum-blossom.svg`
- `bamboo.svg`：OpenClipart，public domain，源文件下载地址：`https://openclipart.org/download/298386/bamboo.svg`
- `clouds.svg`：OpenClipart，public domain，源文件下载地址：`https://openclipart.org/download/314564/clouds.svg`
- `ink-mountain.svg`：OpenClipart，public domain，源文件下载地址：`https://openclipart.org/download/7490/mountain.svg`
- `orchid.svg`：SVG Repo，CC0，源页面：`https://www.svgrepo.com/svg/136382/orchid`
- `chrysanthemum.svg`：SVG Repo，CC0，源页面：`https://www.svgrepo.com/svg/20223/chrysanthemum`

## 使用位置

- 首页 Hero 区：水墨山形、云纹、梅、兰、竹、菊装饰
- 二维码入口区：梅兰竹菊四个图标卡片
